# JK Cars - Rent-a-Car & Excursion Booking Website

## Project Overview
A complete, responsive, futuristic rent-a-car and excursion booking website for a Tunisian agency based in Hammamet, operating across all of Tunisia.

## Design
- **Color Palette**: Electric Blue (#007BFF) and Sky Blue (#00C2FF) on white background
- **Typography**: Manrope font family
- **Style**: Clean, minimal, futuristic with glassmorphism effects
- **Animations**: Framer Motion for smooth page transitions and interactions

## Tech Stack
- **Frontend**: React + TypeScript + Vite
- **Styling**: TailwindCSS + shadcn/ui components
- **Routing**: Wouter
- **Animations**: Framer Motion
- **Backend**: Express.js (currently serving mock data)

## Pages
1. **Home** - Hero section, features, featured cars carousel
2. **Fleet** - Car grid with advanced filters (brand, transmission, fuel, price)
3. **Car Details** - Photo gallery, specs, add-ons, booking panel with dynamic pricing
4. **Excursions** - Destination cards with booking modal
5. **Airport Transfers** - Simple booking form for airport pickups/drop-offs
6. **About** - Agency story, values, coverage map
7. **Contact** - Contact form, map, business info
8. **Booking** - Multi-step booking flow with progress indicator

## Features
- Responsive design (mobile-first)
- Advanced filtering and search
- Dynamic price calculations
- Image galleries with thumbnails
- Smooth page transitions
- Interactive booking forms
- Real-time form validation

## Mock Data
- 6 premium cars with full specifications
- 6 curated excursions across Tunisia
- All data stored in JSON files for easy updates
