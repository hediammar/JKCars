## Project Description

A comprehensive car rental and excursion booking platform for a Tunisian agency, featuring modern UI/UX, real-time booking system, admin dashboard, and multilingual support.

**Challenge**: Modernize booking operations with a scalable, user-friendly platform  
**Solution**: Full-stack React application with Supabase backend integration

---

## Technical Stack

### Frontend
- React 18 + TypeScript + Vite
- TailwindCSS + shadcn/ui
- Framer Motion (animations)
- React Query (state management)
- Wouter (routing)

### Backend
- Supabase (PostgreSQL database)
- Express.js (optional server)
- Drizzle ORM
- Supabase Auth

### Key Libraries
- React Hook Form + Zod (forms/validation)
- jsPDF (PDF generation)
- date-fns/dayjs (date handling)
- Lucide React (icons)
